# User Profile: Interior Designer

The interior designer is the core user of Spaceup.ai, leveraging an agentic, infinite canvas workspace to move from initial site assessment to final high-fidelity renders without the overhead of traditional 3D modeling.

## 🎨 Interior Designer’s Toolkit

An interior designer in Spaceup.ai is familiar with a tiered set of tools, starting from geometric basics to advanced AI-driven spatial orchestration.

### 1. Foundational Tools
These are the daily drivers for workspace management and basic manipulation.

*   **Select (Single & Multi)**: The primary tool for interacting with canvas nodes, images, and connection lines.
*   **Move / Drag**: Free-form positioning of assets on the infinite visual sandbox.
*   **Duplicate / Clone**: Instantly replicating a design node to test a "what-if" scenario without losing the current progress.
*   **Delete**: Removing iterations, redundant reference images, or outdated moodboard assets.
*   **Undo / Redo**: The essential safety net for rapid, experimental design changes.
*   **Pan & Zoom**: Navigating the expansive, multi-branching Design Tree.
*   **Add Image**: The entry point for all designs—uploading site photos, floor plans, napkin sketches, or inspirational imagery.

### 2. Contextual & Spatial Tools
Tools used to define the relationship between aesthetics and physical space.

*   **Connection Lines**: Drawing visual links between context (moodboards) and targets (interior views). In Spaceup.ai, "connection equals context."
*   **Moodboarding**: Grouping fabric swatches, furniture inspiration, and lighting styles to inform the AI's aesthetic direction.
*   **Material Application**: Dragging and dropping specific material finishes (e.g., lime concrete, oak timber) onto design nodes to trigger targeted updates.
*   **Measurement & Scale**: Using reference points to ensure furniture and architectural elements respect real-world proportions.

### 3. Agentic & Iterative Tools
Advanced tools that leverage the Nano Banana AI pipeline for high-speed design development.

*   **Selection Fill (Inpainting / Masking)**: Highlighting a specific area (like a sofa, wall, or ceiling) to modify it while keeping the rest of the structural integrity intact.
*   **Branching (Design Tree)**: Formally starting a new aesthetic direction (e.g., "Minimalist" vs. "Industrial") from a shared base image.
*   **Conversational Command**: Interacting with the Spaceup Agent to orchestrate complex visual changes through natural language.
*   **Z-Depth Isolation**: Defining foreground and background layers to prevent the AI from overwriting structural structural architectural details during furniture swaps.

### 4. Collaboration & Delivery Tools
Tools focused on the feedback loop and final output.

*   **Spatial Commenting**: Pining notes and feedback directly onto specific coordinates of a design iteration.
*   **Comparison (A/B Testing)**: Side-by-side evaluation of different branches in the Design Tree.
*   **Direct Share**: Generating view-only links for clients to experience the design journey on the canvas.
*   **High-Res Export**: Finalizing a conceptual iteration into a professional-grade render for specification sheets.
