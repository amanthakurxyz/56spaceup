# Future Roadmap - Spaceup.ai

### Deferred Features
- [ ] **Scale Ruler**: A simple ruler that interprets the depth/scale of the image to show rough real-world dimensions.
- [ ] **Material Eyedropper**: Sample colors or materials from nodes to use as reference.
- [ ] **Clear Canvas**: A "Reset Project" action to wipe all nodes and edges (requires confirmation).
- [ ] **Fit to Screen**: Auto-center and zoom to show all design nodes.
- [ ] **Grid Toggle**: Option to hide/show the background dot grid for a cleaner presentation.
- [ ] **Magnetic Grids**: Snapping nodes to a layout hierarchy.
